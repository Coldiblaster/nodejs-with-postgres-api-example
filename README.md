### Requirements

* Node.js v8+
* Postgres running on local instance or Docker

### Running on localMachine

* Install dependencies - `npm i`
* Build typescript - `npm run build`
* Run project - `npm start`

### Docker

* `docker-compose up`

### Documentation

* Go to swagger page - `localhost:3000/documentation`
